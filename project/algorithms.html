Assignment problems are quite common in real world applications. The idea is to assign a few entities (factory, electronic components etc.) to several positions. Each assignment incurs an activity cost among entities, called flow; and distance cost (logistics, latency etc.); and the sum of product of both costs constitutes the total cost. This is known as a Quadratic Assignment Problem (QAP). Examples of QAP are, but not limited to:

  • Factory location assignment
  • Hospital layout
  • Airport gate assignment
  • Electrical wiring planning
  • Scheduling

QAP is an optimization problem and the goal is to find a permutation giving the lowest total cost (objective function). In this project, a QAP solver using heuristics and metaheuristics was developed as a library in C++. Like every optimization problem, this QAP solver first generate an initial solution, then iteratively compute the next solution given the heuristics till it reaches a termination cue. Each choice of heuristics was able to reach global optimum depending on the difficulty.

List of initial solution generator:

  • Iterative random generator
  • Greedy generator

List of heuristics / metaheuristics:

  • Local search with dynamic neighborhood operators
  • Simulated annealing
  • Genetic algorithm
  • Tabu search

The QAP solver tackles problems from the QAPLIB website.

Check out the repository of the QAP solver: [GitHub] open_in_new
