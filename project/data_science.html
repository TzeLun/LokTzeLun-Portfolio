Predicting the Landing Success of Rockets for Reusability: A Case Study Featuring SpaceX

Executive summary: This report aims to address whether it is possible to predict if the rocket will successfully land on the first stage of launch so it could be reusable with past data instead of scientific simulation. REST APIs and web-scraping were used for data collection, primarily for the launch specification and launch outcomes (success or failure). A detailed Exploratory Data Analysis (EDA) was conducted to study the correlation between independent variables and their influence on the launch outcomes. Interactive maps and dashboard were created to understand the geography of launch sites and other launch success factors. Four classification models: K-Nearest Neighbors (KNN), Support Vector Machine (SVM), Logistic Regression (LogReg), and Decision Trees (Tree) were trained with cross validation via a grid search respectively, to find their best performing parameters. Through our analysis, Decision Trees was the best classifier with a final test accuracy of 83.33% and a precision score of 0.917.

Data collection:

  • Using SpaceX's REST API to obtain past data on launch specifications, ie.: rocket booster version, payload mass, launch site etc. These are the independent variables for predicitive analysis.

  • Web-scraping with BeautifulSoup module to extract past data of launch outcomes unavailable from the SpaceX's website. The launch outcomes (success or failure) are the class labels to predict.

Data wrangling:

  • Processed data into Panda's dataframe and final clean ups: replacing missing data with mean values and correcting false data types.

  • Plotted categorical, scatter, and bar charts to understand the correlation between variables used for prediction via Seaborn and Matplotlib.

EDA - Dashboards:
Constructed with Plotly Dash, a dashboard library in Python

  • Devised a pie chart dashboard which users could selectively choose to display the percentage of launch successes contributed by all launch sites or the proportion of success and failure for a single launch site via a dropdown menu.
  • Devised a scatter plot of different rocket booster versions against the payload mass. Equiped with a sliding bar to adjust the payload range for finer analysis.

Model training and evaluation:

  • Developed the 4 machine learning models (KNN, LogReg, SVM, Tree) with scikit-learn. Discovered the best hyperparameters for each model using grid search.

  • Compared and evaluated the model performance based on their confusion matrix and test accuracy.

Results of predictive analysis:
All 4 models achieved the same test accuracy of 83.33%. However, decision trees had the least number of false positives, hence it has a higher precision score of 0.917 while the rest scored 0.2. Conclusion, decision tree is the best classifier (in this case) to predict the launch outcomes of SpaceX rockets.

Synthetic Minority Oversampling Technique (SMOTE) for Imbalance Data Correction

A dataset is imbalance when the number of data classes are disproportionate. SMOTE works by synthesizing data of the minority class labels to augment the dataset. New data is created by adding a randomly picked sample with the differences between its nearest neighbors. Check out the Python implementation at [GitHub] open_in_new
